# DM message template

Replace the placeholders below with your own copy, then save this file at the
path you pass to the skill as `MESSAGE_PATH`. Plain text only; emoji are fine.

Keep the message under roughly 500 characters. Rednote DM previews truncate
aggressively, and very long messages read as bot-like.

```
Hi there!

I'd love to share {{ONE_LINE_HOOK}}: {{LINK_OR_TEASER}}

{{TWO_OR_THREE_SENTENCE_PITCH}}

About us: {{ONE_LINE_ABOUT}}

{{SOFT_CALL_TO_ACTION}}
```

## Tips

- Open with a personal hook, not a sales line.
- Make the first paragraph readable in the truncated preview.
- One link maximum. More than one triggers spam flags.
- Sign off as a person, not a brand account name.
- Test the message on yourself first by DMing your own alt account.
